PROCESO DE INSTALACION

Como primer paso principalmente estuvimos limpiando algunos contenedores para asi poder contener el agua , que quedaron totalmente limpios para el uso que se les va dar a cada uno de ellos.

Por otro lado tambien estuvimos haciendo una limpieza como desquelitando y escarbando para asi poder tener la tierra un poco mas pareja,tambien se hizo un agujero en el cual su profundidad fue de aproximadamente 40cm y de 1x5metros cuadrados.

Para su uso del agujero que se hizo se puso un plastico de maxima dureza donde se va introducir,el agua con peces y los cultivos que se van sembrar.

Se agregaron algunos bloques de semento y se les puso un tipo de mezcla para asi poder pegar los bloques de cemento y poder seguir con el procedimiento de filtracion.

Se comenzo a tejer una malla(malla gallinera)hasta poder completarla y tambien estuvo otra malla (malla electrosoldada)que media 13 metros de largo y 1 metro de ancho.

En los tiempos libres pasabamos a desquelitar todo lo que hierba otros tipos de hierba y se junto todo el desecho de lo que trabajamos para asi ir a poderlo desechar en el lugar donde se tira donde es llamado la nopalera.

Despues se comenzo hacer el filtro del agua se introducio tierra en las orillas para asi comenzar a colocar la malla gallinera ,introducimos tierra en el centro del filtro de agua colocando el soporte dentro dela malla para cubrir y colocar tierra dentro de este mismo.

Se emparejo para poner bloques y asi poder subir los contenedores donde va ir introducida el agua y comenzar con el proceso de tilapias.

Se agregaron lonas y tambien piedras arriba del filtro de agua asi de tal manera que tambien colocamos algunas botellas en las puntas de las varillas de los contenedores.

Se separo la piedra que estaba en mejor estado y la otra se desecho para asi poder ponerla en el filtro del agua.

Se agregaraon las piedras la filtro de agua y se le agrego agua al agujero que se creo para asi darle procedimento al metodo de flitracion.

Como ultimo paso los ingenieros autorizados instalaron los tubos para poder comenzar todo el proceso del proyecto y a darle proceso a todo el cultivo de plantas y el proceso de tilapias.

Se agrego la piedra que se estaba separando para poder y comenzar allenar sobre el cuadro que se escarbo y se coloco el plastico de maxima dureza y se coloco la piedra dentro de el para haci dar un comienzo a la filtracion donde va y colocado lo de las plantas de reproduccion.

Se comenzaron allenar los contenedores de agua luego se comenzo a meter un filtro de agua para poder pornerla a una un cierto grado para que no se mureran las tilapias tambien se agrego una metodo de oxigeno para tenerlos a temperatura ambiente y a hora si dar "el procedimiento a todo el proceso que es llamado "ACUAPONIA".

Asi quedo ya por terminado todo el proceso de filtracion tanto como de plantas como de peces. "ACUAPONIA"
